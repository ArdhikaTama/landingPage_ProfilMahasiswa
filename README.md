Landing Page Profil Mahasiswa
## Deskripsi Proyek
Landing page profil mahasiswa modern dan responsif yang dibuat menggunakan HTML, CSS, dan JavaScript. Program ini dirancang untuk menampilkan profil, pendidikan, keterampilan, dan portfolio mahasiswa dengan tampilan yang menarik

## Fitur Utama
Desain Responsif - Optimal di desktop, tablet, dan mobile
Dark/Light Mode - Toggle tema gelap dan terang
Animasi Scroll Reveal - Elemen muncul dengan animasi saat di-scroll
Filter Portfolio - Filter proyek berdasarkan kategori
Validasi Form Kontak - Validasi client-side untuk form kontak
Smooth Scrolling - Navigasi halus antar section
Mobile Navigation - Menu hamburger untuk perangkat mobile
Modal Interaktif - Detail portfolio dalam modal popup

## Teknologi yang Digunakan
HTML5 - Struktur semantik
CSS3 - Styling dengan Flexbox & Grid
JavaScript (ES6+) - Interaktivitas

## Warna Utama
Primary: #1a73e8 (Biru)
Background: #ffffff (Putih) / #f8f9fa (Abu-abu muda)
Text: #202124 (Gelap) / #5f6368 (Abu-abu)

## Struktur Halaman
Header/Navbar - Navigasi dengan sticky position
Hero Section - Foto profil dan introduction
Tentang Saya - Deskripsi dan informasi pribadi
Pendidikan - Timeline pendidikan visual
Skill/Kemampuan - Progress bars untuk skills
Portfolio - Grid proyek dengan filter
Kontak - Form kontak dengan validasi

## Fitur Interaktif JavaScript
Smooth Scrolling - Navigasi halus ke section
Active Navigation - Highlight menu sesuai section yang aktif
Theme Toggle - Switch antara dark/light mode
Portfolio Filter - Filter proyek berdasarkan kategori
Form Validation - Validasi input form kontak
Mobile Menu - Toggle menu untuk perangkat mobile
Scroll Animation - Animasi saat elemen masuk viewport
